# Outlook 365加载项现已可用

适用于Outlook 365的Adobe Workfront加载项已于2017年5月16日推出。

通过此加载项，您可以直接从Web上的Outlook 365访问Workfront功能，以及使用Office 365订阅随附的Windows和Mac桌面版Outlook时。

>[!NOTE]
>
>此加载项不再可用。
