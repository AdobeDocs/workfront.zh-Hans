# 在23.3版本发行时间范围内进行外观更新

本页介绍了在23.3版本时间范围内对Adobe Workfront应用程序各个区域的外观进行的细微更新。 在发布到“预览”后，生产环境中将至少在2周内提供这些增强功能。

有关23.3版本中可用的所有更改的列表，请参阅 [23.3版本概述](/help/quicksilver/product-announcements/product-releases/23.3-release-activity/23-3-release-overview.md).
