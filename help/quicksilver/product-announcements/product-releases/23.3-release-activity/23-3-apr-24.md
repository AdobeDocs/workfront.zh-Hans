# 23.3 2023年4月24日的发行活动周

本页介绍2023年4月24日这一周23.3版本对预览环境所做的所有增强。 23.3版本的生产环境中将提供这些增强功能。

有关23.3发行周期中此刻可用的所有更改的列表，请参阅 [23.3版本概述](/help/quicksilver/product-announcements/product-releases/23.3-release-activity/23-3-release-overview.md).

## 添加到动态展示板模板的简单过滤器

>[!NOTE]
>
>此功能仅通过Workfront主板的早期功能选择加入来提供。

动态展示板模板上的过滤器已得到简化，以便您更快地创建展示板。 可用的过滤器包括Workfront项目和按团队或用户的分配。 您可以根据需要切换到高级过滤器。 动态展示板的“配置”面板中也提供了这些过滤器选项。

有关更多信息，请参阅 [创建或编辑展示板](/help/quicksilver/agile/get-started-with-boards/create-edit-board.md).

## 使用Adobe Workfront调整自动校样工作流模板设置以进行Creative Cloud

现在，您可以直接在Creative Cloud中调整现有的自动化工作流模板设置。 选择现有的自动化工作流模板后，您可以：

* 禁用阶段
* 添加其他收件人
* 更改校样角色
* 调整截止时间
* 更新电子邮件通知
* 更多！

有关更多信息，请参阅 [使用上传文档和校样 [!DNL Adobe Workfront] 插件 [!DNL Creative Cloud] 应用程序](/help/quicksilver/workfront-integrations-and-apps/adobe-workfront-for-creative-cloud/wf-cc-docs-proofs-toc.md).

以下增强功能可用于以下Creative Cloud应用程序：

* Photoshop
* XD
* InDesign
* Illustrator
