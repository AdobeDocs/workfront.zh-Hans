## 正在删除的功能

### 从生产环境中删除

* [删除Workfront中的背书功能](../../../../product-announcements/product-releases/quarterly-release-archive/2017.2-release-activity/2017.2-beta-1-release-activity.md#deprecating-the-endorsements-functionality-in-workfront)

### 从抢先访问中删除

“预览”和“生产”环境中都弃用了“抢先访问”。

由于进行了这项更改，2017.2测试版1版本的“预览”环境中的“抢先体验”中删除了以下功能。 

* [通过拖放对任意列表中的列重新排序（功能将被删除）](../../../../product-announcements/product-releases/quarterly-release-archive/2017.2-release-activity/2017.2-beta-1-release-activity.md#reorder)

此功能将在2017.2版本的生产环境中从“抢先体验”中删除，任何用户都将无法使用此功能。
