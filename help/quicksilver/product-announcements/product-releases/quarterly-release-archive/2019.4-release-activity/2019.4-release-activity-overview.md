## 增强了所有用户

2019.4版本之前已发布一些功能。 要查看2019.4版本之前发布的内容，请参阅 [2019.4其他增强功能](../../../../product-announcements/product-releases/quarterly-release-archive/2019.4-release-activity/2019.4-other-enhancements.md).

2019.4版本包含以下增强功能：

* [列表增强功能](#list-enhancements)
* [移动设备增强功能](#mobile-enhancements)
* [资源计划增强功能](#resource-scheduling-enhancements)
* [其他增强功能](#other-enhancements)
