# 2017.3发布活动概述

本页介绍了2017.3版本中包含的功能。 

此页面上描述的功能当前在“预览”环境中可用，并且将在11月的“生产”环境中可用。  

>[!IMPORTANT]
>
> 在生产环境中发布之前，预览环境中的功能可能会发生更改。

2017.3版本包含面向Workfront管理员和其他用户的增强功能。

## 移动设备增强功能 {#mobile-enhancements}

>[!NOTE]
>
>此部分中描述的功能现在可以与您的Workfront生产帐户一起使用；这不是“预览”功能。

2017年8月，适用于Android和iOS的Workfront移动设备应用程序的生产版本中添加了以下功能：

* 提交请求
* 工时表新条目
* 自定义表单编辑
* 校对审批请求
* 订阅项目的推送通知

有关这些更改的详细信息，请参阅 [更新了适用于iOS和Android的移动应用程序（2017年8月初）](../../../announcements/announcement-archive/updated-mobile-app.md).

以下附加功能将在11月8日生产版本发布之前提供：

* 访问项目列表
* 访问项目中的任务和子任务的列表
* 访问项目的问题列表
* 在项目上记录新问题
