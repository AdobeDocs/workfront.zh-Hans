# Adobe Workfront有限可用功能的2026年第二季度发布活动

本文介绍了在2026年第二季度发布的Workfront功能的有限可用性。

<!--keep the sentence below for all future quarterly release pages-->

有关为Adobe Workfront发布的所有有限可用性功能的列表，请参阅[Adobe Workfront有限可用性功能发布活动：文章索引](/help/quicksilver/product-announcements/product-releases/limited-availability-release-activity/limited-availability-release-activity-article-index.md)。

## 有限可用性功能的发行说明

此部分包含新功能的发行说明。

### 更新了执行单次或批量分配任务时的体验

>[!NOTE]
>
>生产发布日期： 2026年2月12日

在编辑单个任务或批量编辑这些任务时，我们更新了“编辑任务”框中的“工作总揽”部分。

为任务分配资源提供了新的体验。 现在，在编辑一个任务或编辑多个任务时，都可以使用此功能。 我们已将以下字段添加到“工作总揽”部分：

* 分配
* 所有者或任务所有者
* 受让人的角色

有关信息，请参阅[编辑任务](/help/quicksilver/manage-work/tasks/manage-tasks/edit-tasks.md)。

### 保留人员配备计划的记帐费率

>[!NOTE]
>
>生产发布日期： 2026年3月5日

我们已在“人员配备计划详细信息”页面的“财务”部分添加了&#x200B;**保留帐单**&#x200B;字段。

当此标志设置为False（关）时，不会保留记帐费率，并且费率层次结构用于记帐费率计算。

当此标志设置为True （开）时：

* 人员配置计划上所分配资源的当前记帐费率将保留，层次结构中费率的任何变动都不会反映在人员配置计划的“资源”区域。
* 如果向“资源”表添加新行，则保留来自记帐费率层次结构的初始记帐费率。
* 如果用户在启用标记之前手动覆盖了记帐费率值，则覆盖率将保留。 启用此标志后，不允许手动覆盖记帐费率。

激活保存后，无法将其关闭。

在复制启用了标志的人员配备计划时，系统会自动关闭此标志。 如果新人员配备计划需要此标志，资源经理必须将其打开。
