# 23.1发行版概述

本页介绍了23.1版本中包含的功能。

这些增强功能在2023年1月18日和19日的23.1版本的生产环境中提供。

23.1版本网络研讨会于2023年1月10日星期二举行。 您可以[在此处](https://event.on24.com/wcc/r/3995983/A5FC731C36313CE081AEBB3B1AA39C77)注册网络研讨会以观看即时录制。

<span class="preview">非周期功能（在23.1发布日期之前发布到生产环境的功能）以黄色突出显示。</span>

### 即将从Workfront中删除的功能

以下功能即将从Workfront中删除：

#### **弃用并从预览环境中删除营销活动(Beta)区域**

我们将于2023年1月9日从“预览”环境中移除“营销活动” (Beta)区域。 我们的产品团队将审核此功能并将其包含在未来的产品版本中。

目前，我们正在做以下更改：

* 从主菜单和预览环境的布局模板中删除营销活动(Beta)区域。
* 从主菜单和生产环境中的布局模板中，为那些通过请求启用了促销活动(Beta)的客户删除了促销活动区域。

此外，所有描述此功能的文章也将被删除。

#### **列出控件（来自设置中的接口区域）**

在23.1版本中，将删除“设置”>“接口”中的列表控件。

某些列表控制功能正在移至“设置”>“界面”>“筛选器/视图/分组”区域。 有关详细信息，请参阅[创建、编辑和共享默认筛选器、视图和分组](/help/quicksilver/administration-and-setup/set-up-workfront/configure-system-defaults/create-and-share-default-fvgs.md)。

#### 适用于Illustrator和InDesign的&#x200B;**Workfront扩展**

我们正在使用更新的Creative Cloud插件替换Illustrator和InDesign的[Workfront扩展](/help/quicksilver/documents/workfront-for-adobe-creative-cloud/use-wf-adobe-cc.md)。 从2022年末开始，此扩展将不再可用于在Adobe Exchange中安装。

此扩展中的功能以及几个增强功能在更新的Adobe Workfront的Creative Cloud插件中提供：
* [适用于Photoshop的Adobe Workfront](/help/quicksilver/workfront-integrations-and-apps/adobe-workfront-for-creative-cloud/wf-cc-install-ps.md)
* [适用于XD的Adobe Workfront](/help/quicksilver/workfront-integrations-and-apps/adobe-workfront-for-creative-cloud/wf-adobe-xd-install.md)
* [Adobe Workfront设计和视频](/help/quicksilver/workfront-integrations-and-apps/adobe-workfront-for-creative-cloud/wf-install-cc.md)

#### **资源计划程序**

>[!NOTE]
>
>10月20日，从预览环境中删除了资源计划程序。 该版本将于2023年1月从23.1版本的生产环境中删除。


作为弃用资源计划工具的一部分，我们删除了以下区域：

* 主菜单的资源区域中的“计划”部分

* 团队页面的“计划”部分

* 项目页面的“计划”部分

工作负载均衡器正在替换资源计划程序。

#### **事件订阅API中的SHARE事件类型。**

>[!NOTE]

>  此功能在创建新订阅时已被弃用，并将于2023年1月被删除。

此事件类型允许客户观察对象的权限何时更新。

有关事件订阅以及将继续受支持的其他事件类型的详细信息，请参阅[事件订阅API](/help/quicksilver/wf-api/general/event-subs-api.md)
