# 2024年第三季度集成增强功能

本页介绍了在2024年第三季度版本中对“预览”环境进行的所有集成增强。 这些增强功能将在2024年第三季度版本的生产环境中提供。

有关2024年第三季度发布周期中此时可用的所有更改列表，请参阅[2024年第三季度发布概述](/help/quicksilver/product-announcements/product-releases/24-q3-release-activity/24-q3-release-overview.md)。

## Workfront for Experience Manager Assets和Assets Essentials改进

>[!NOTE]
>
>预览版本： 2024年6月27日；面向所有客户的生产：使用24.7版（2024年7月18日）。

我们对Workfront进行了以下改进，以便实现Experience Manager Assets和Assets Essentials集成：

* 该集成现在支持GCP作为云服务提供商。 AWS和Azure以前也受支持。 有关详细信息，请参阅[Adobe Experience Manager Assets集成概述](/help/quicksilver/documents/adobe-workfront-for-experience-manager-assets-essentials/aem-asset-integrations.md)。

* 通过集成发送到Experience Manager的文件的大小限制已增加到30 GB。 以前，限制为5 GB。 有关详细信息，请参阅[将文档发送到Experience Manager Assets或Assets Essentials](/help/quicksilver/documents/adobe-workfront-for-experience-manager-assets-essentials/send-to-aem.md)。
