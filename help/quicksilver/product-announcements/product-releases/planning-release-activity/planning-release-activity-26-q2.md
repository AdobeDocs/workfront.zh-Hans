# Adobe Workfront Planning 2026年第二季度发布活动

本文介绍了在2026年第二季度版本中为Workfront计划发布的功能。

<!--keep the sentence below for all future quarterly release pages-->

有关为Adobe Workfront Planning发布的所有功能的列表，请参阅[Adobe Workfront Planning发布活动：文章索引](/help/quicksilver/product-announcements/product-releases/planning-release-activity/planning-release-activity-article-index.md)。

## 在Workfront Planning的用户预输入字段中显示电子邮件

>[!NOTE]
>
>预览： 2026年1月22日
>生产快速发布： 2026年2月12日
>适用于所有人的生产： 2026年4月16日

现在，当您将用户添加到以下区域时，会显示用户的电子邮件：

* 人员类型计划字段
* 在共享Planning对象时的“共享”框中

在此增强功能之前，将用户添加到这些区域时，除了用户名之外，仅显示用户的主要角色。
有关更多信息，请参阅以下文章：

* [创建字段](/help/quicksilver/planning/fields/create-fields.md)
* [共享工作区](/help/quicksilver/planning/access/share-workspaces.md)

## 人员字段已作为Planning连接的查找字段启用

>[!NOTE]
>
>预览： 2026年1月14日
>生产快速发布： 2026年2月12日
>适用于所有人的生产： 2026年4月16日

现在，在连接两个Planning记录类型时，可以将“人员”字段添加到查找字段。

有关信息，请参阅[连接记录类型](/help/quicksilver/planning/architecture/connect-record-types.md)。
