# Adobe Workfront Proof 23.3发布活动： 2023年6月5日开始的一周

本页介绍了23.3版在2023年6月5日发行的Adobe Workfront Proof对预览环境所做的所有增强。

除非另外指定，否则这些增强功能将在23.3版的生产环境中可用。

## 更改对Microsoft Office校样的支持

为维护强大且可扩展的校样处理解决方案，我们正在更改受支持的Microsoft Office文件类型和文件大小限制。

不再支持以下Office文件类型作为校样：
XLT， XLTX， DOT， DOTX，锅， POTX， PPS， PPSX

用户可以继续创建以下Office文件类型的验证：
DOC、DOCX、PPT、PPTX、XLS、XLSX

PowerPoint和Word文件大小限制现在为512MB。

有关详细信息，请参阅[支持的校对文件类型和大小限制概述](/help/quicksilver/review-and-approve-work/proofing/proofing-overview/supported-proofing-file-types.md)。
