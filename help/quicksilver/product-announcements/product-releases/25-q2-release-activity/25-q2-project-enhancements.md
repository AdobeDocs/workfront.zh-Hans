# 2025年第二季度项目改进

本页介绍了在预览环境2025年第二季度版本中所做的所有项目增强功能。 如上所述，这些增强功能将在生产环境中提供。

有关2025年第二季度发布周期中此时可用的所有更改的列表，请参阅[2025年第二季度发布概述](/help/quicksilver/product-announcements/product-releases/25-q2-release-activity/25-q2-release-overview.md)。

## 在“编辑项目”框中编辑项目时，向项目添加注释

>[!NOTE]
>
>预览版本：2025年2月13日；生产版本快速发布：25.3版（2025年3月）；生产版本季度发布：25.4版（2025年4月）

现在，您可以在编辑项目框中编辑项目时向项目添加注释。 批量编辑多个项目时，您还可以一次向这些项目添加注释。 在此更新之前，编辑项目时此功能不存在。

有关信息，请参阅[编辑项目](/help/quicksilver/manage-work/projects/manage-projects/edit-projects.md)。
