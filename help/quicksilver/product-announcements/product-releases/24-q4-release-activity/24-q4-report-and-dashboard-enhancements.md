# 2024年第四季度报告和控制板增强功能

本页介绍了在预览环境2024年第四季度版本中所做的所有报告和控制板增强。 如上所述，这些增强功能将在生产环境中提供。

有关2024年第四季度发行周期中此时可用的所有更改的列表，请参阅[2024年第四季度发行概述](/help/quicksilver/product-announcements/product-releases/24-q4-release-activity/24-q4-release-overview.md)。

## Workfront Data Connect可用于新计划

>[!NOTE]
>
>适用于所有客户的生产版本：随24.10版本（2024年10月17日）一起提供
>
>_仅适用于使用某个新Adobe Workfront计划的组织。 Data Connect已包含在Ultimate计划中，并将于2025年上半年作为Prime和Select计划的附加产品提供。_

Workfront Data Connect将可供使用某个新Workfront计划的客户使用。 Data Connect允许组织将其数据作为一个安全且可扩展的数据湖进行访问，该数据湖可以使用业务智能工具进行分析和可视化，或者存储在外部。 此外，组织可以使用Data Connect查看以前不可用的数据分析，例如基于时间的趋势分析、变量映射，以及结合Workfront数据对外部系统数据进行分析。

请联系您的客户代表，详细了解您当前的Adobe Workfront计划以及如何购买插件。 有关Workfront Data Connect的详细信息，请参阅[Workfront Data Connect概述](/help/quicksilver/reports-and-dashboards/data-lake/data-lake-overview.md)。
