### API版本21

Workfront API版本21于2025年10月23日发布。 对于API版本21，我们修改了一些资源和端点。 有些更改支持新功能，有些更改使您能够更轻松地使用通过API提供的信息。

>[!IMPORTANT]
>
>此API版本更改具有一项重大更改，可能会影响现有API调用。 这是因为API版本21使用事件订阅版本2。
>
> For multi-select fields, Event Subscriptions version 2 always sends as an array. 如果选择了多个值，则版本1会发送一个数组。 If only one value was selected, it sent a string.

有关新增功能和更新的信息，请参阅[API版本21](/help/quicksilver/wf-api/api/new-api-version-21.md)中的新增功能。

有关API版本的信息，请参阅[API版本控制和支持计划](/help/quicksilver/wf-api/api/api-version-support-schedule.md)。
