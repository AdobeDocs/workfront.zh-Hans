# 2026年第三季度请求增强功能

本页介绍了在2026年第三季度版本中对“预览”环境所做的请求增强。 如上所述，这些增强功能将在“生产”环境中提供。

有关2026年第三季度发布周期中此时可用的所有更改列表，请参阅[2026年第三季度发布概述](/help/quicksilver/product-announcements/product-releases/26-q3-release-activity/26-q3-release-overview.md)。

## 旧请求体验和新请求体验之间的匹配请求状态

>[!NOTE]
>
>预览： 2026年5月27日生产快速发布： 2026年6月11日适用于所有人的生产： 2026年7月16日

为了获取有关您的请求处于正确状态的更多上下文，我们将在新请求体验中更新请求的状态。

对于以下状态，请求将在新请求体验和旧请求体验中显示相同的状态：

* 新建
* 暂停

在此更新之前，如果请求在旧版请求体验列表中处于“新建”或“已搁置”状态，则会在新请求体验列表中显示为“进行中”。 现在，这两个列表的状态相匹配。

作为此更新的一部分，“已完成”状态也已重命名为“已完成”。

有关详细信息，请参阅[查看已提交的请求](/help/quicksilver/manage-work/requests/create-requests/locate-submitted-requests.md)。

## 在“请求”区域的请求列表中添加了预定义视图，以及我的请求小组件

>[!NOTE]
>
>预览： 2026年4月23日生产快速发布： 2026年4月23日适用于所有人的生产： 2026年4月23日超出计划&rbrack;{type=Neutral}

为了帮助您更好地组织请求列表，我们已将以下预定义视图添加到请求区域的请求列表和主页中的我的请求构件：

* 我的请求
* 我的未结请求
* 我的草稿
* 打开请求。 在“我的请求”小组件中不可用。

Workfront管理员可以在布局模板中添加或删除视图。 您无法编辑、删除或共享这些视图。 您可以复制它们并更新或共享副本。

有关信息，请参阅[在请求区域](/help/quicksilver/manage-work/requests/create-requests/create-views-for-requests-list.md)中创建和管理视图。
