# Adobe Workfront Fusion发行版活动概述

Adobe Workfront Fusion可将您的应用程序和Web服务无缝连接，以使您能够专注于新任务，而不是一次又一次地重复相同的任务。

有关如何使用Workfront Fusion的信息，请参阅[Adobe Workfront Fusion](../../../workfront-fusion/workfront-fusion-2.md)。
