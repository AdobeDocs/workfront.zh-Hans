# Workfront Fusion版本活动： 2022年2月28日开始的一周

本页介绍2022年2月28日这一周Adobe Workfront Fusion中所做的所有增强功能。

有关所有最近更改的列表，请参阅 [Adobe Workfront Fusion版本活动](../../../product-announcements/product-releases/fusion-release-activity/fusion-release-activity.md).

有关Workfront Fusion中最近的错误修复列表，请参阅 [Workfront维护更新](https://one.workfront.com/s/article/Workfront-Maintenance-Updates-1882317350) 页面，并检查标记为Workfront Fusion维护更新的任何更新。

## 在Azure DevOps >自定义API调用模块中选择API版本

为了让您能够更好地控制Workfront Fusion的Azure DevOps连接器中的API调用，我们在自定义API调用模块中添加了选择Azure DevOps API版本的功能。 Workfront Fusion支持Azure DevOps API版本4.1至7.1。

以前，您无法选择API版本。

有关更多信息，请参阅 [Azure DevOps模块](../../../workfront-fusion/apps-and-their-modules/azure-dev-ops.md).
