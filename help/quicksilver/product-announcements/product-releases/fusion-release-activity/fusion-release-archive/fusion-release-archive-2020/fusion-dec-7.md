# Workfront Fusion版本活动： 2020年12月7日开始的一周

本页介绍2020年12月7日这一周Adobe Workfront Fusion中所有增强功能。

有关所有最近更改的列表，请参阅 [Adobe Workfront Fusion版本活动](../../../../../product-announcements/product-releases/fusion-release-activity/fusion-release-activity.md).

有关Workfront Fusion中最近的错误修复列表，请参阅 [Workfront维护更新](https://experienceleague.adobe.com/docs/workfront-known-issues/releases/current-updates.html) 页面，并检查标记为Workfront Fusion维护更新的任何更新。

## Jira服务器连接器和模块现已可用

新的Jira服务器连接器现在可用于Workfront Fusion 2.0。Jira服务器连接器提供与当前Jira云连接器相同的功能。 使用Jira服务器模块，您可以：

* 在添加、修改或删除记录时会触发方案
* 创建、读取、更新或删除记录
* 列出或搜索记录
* 下载附件
* 向冲刺添加问题
* 进行自定义API调用

以前，Jira模块仅可用于Jira Cloud。

有关可用Jira模块的更多信息，请参阅 [Jira软件模块](../../../../../workfront-fusion/apps-and-their-modules/jira-software-modules.md).
