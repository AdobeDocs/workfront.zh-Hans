# Adobe Workfront MCP服务器概述

[!DNL Adobe Workfront] MCP（模型上下文协议）服务器允许您在受支持的AI代理平台上通过自然语言对话处理Workfront数据。 您不需要通过Workfront来查找项目、更新任务或检查批准，而是会以简单的英语询问AI代理平台，而AI代理平台会为您工作。

使用Workfront MCP服务器，您不需要知道Workfront API或有关MCP的任何信息。 设置AI代理平台与Workfront进行一次对话，然后开始询问。

## 设置和使用Workfront MCP服务器

在使用Workfront MCP服务器之前，Workfront管理员必须在Workfront中启用MCP服务器访问，并且您必须将AI代理平台连接到Workfront。 有关详细信息，请参阅[配置Adobe Workfront MCP服务器](/help/quicksilver/workfront-basics/workfront-mcp-server/configure-workfront-mcp-server.md)。

设置后，您可以要求AI代理平台查找、创建、更新和管理Workfront项目。 有关可用操作、示例请求和数据处理的详细信息，请参阅[使用Adobe Workfront MCP服务器](/help/quicksilver/workfront-basics/workfront-mcp-server/use-workfront-mcp-server.md)。
