# 对象标题概述

您可以在 [!DNL Adobe Workfront] 查看其标题时。

除了对象的名称之外，标头还可以包含对象的所有者、状态或完成百分比。

[!DNL Workfront] 优先考虑对象的名称，在标头中为对象分配尽可能多的空间。 当对象名称过长时，该对象名称会被截断。 要显示对象的全名，可将鼠标悬停在对象上。

## 访问对象的标头

访问中对象的标头 [!DNL Workfront] 对于具有该对象的所有对象，都是相同的。

例如，要访问项目的标题：

1. 转到项目。\
   标题显示在页面顶部，并包含项目名称。

   ![](assets/project-header-350x18.png)

## [!UICONTROL 主页] 标题概述

主页中提供了以下标头：

* 任务：有关如何使用此标题的更多信息，请参阅 [任务标题概述](#task-header-overview) 在本文中。
* 问题：有关如何使用此标题的更多信息，请参阅 [问题标题概述](#issue-header-overview) 在本文中。

## 可自定义的标题

您的 [!DNL Workfront]  或者，组管理员可以使用布局模板自定义项目、任务和问题的标题。

本文介绍了所有对象的默认标题，包括项目、任务和问题。

有关自定义对象标头信息的信息，请参阅 [使用布局模板自定义对象标头](../../administration-and-setup/customize-workfront/use-layout-templates/customize-object-headers.md).
