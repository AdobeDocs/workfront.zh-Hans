# 查看[!UICONTROL 主页日历]

您可以使用[!DNL Adobe Workfront Home]中的[!UICONTROL 日历]视图在一个位置管理所有分配的工作项。 您的[!UICONTROL 主页日历]视图仅对您可见，只有您才能管理您的[!UICONTROL 主页日历]。

要了解有关主页日历的更多信息，请参阅[主页日历视图](../../../workfront-basics/using-home/using-the-home-area/home-calendar-view.md)。

## 查看[!DNL Workfront] [!UICONTROL 主页日历]

1. 单击Adobe Workfront左上角的&#x200B;**[!UICONTROL 主页]**&#x200B;图标![](assets/home-icon-30x29.png)。

   >[!NOTE]
   >
   >您的[!DNL Workfront]管理员可能会对您环境中的[!UICONTROL 主页]图标进行以下更改：
   >
   >* 请将其替换为用于说明贵组织的自定义图像。 在这种情况下，图标的外观将与本文中显示的有所不同。
   >* 将链接到该页面的页面替换为其他页面。 在这种情况下，请单击页面右上角的&#x200B;**[!UICONTROL 主菜单]** ![](assets/main-menu-icon.png)，然后单击&#x200B;**[!UICONTROL 主页]**。


1. 单击主页左上角的&#x200B;**[!UICONTROL 工作列表]**，然后在下拉菜单中单击&#x200B;**[!UICONTROL 日历]**。

   ![](assets/work-list-calendar-drop-down-expanded-home-nwe.png)

   显示[!UICONTROL 主页日历]视图。

   ![](assets/calendar-group-by-drop-down-home-nwe-350x135.png)

   分配给您或分配给您的组、团队或角色的所有工作项都将显示在左侧面板中。

   您可通过将任务从左侧面板拖放到日历中来管理工作日。 要了解如何填充[!UICONTROL 主页日历]，请参阅[使用[!UICONTROL 主页日历]视图](../../../workfront-basics/using-home/using-the-home-area/use-home-calendar-view.md)。
