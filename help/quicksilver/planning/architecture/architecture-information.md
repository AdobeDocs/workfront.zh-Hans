# 架构：文章索引

{{planning-important-intro}}

作为营销运营负责人，您可以使用Adobe Workfront Planning为所有团队在营销生命周期中组织工作。

使用Workfront Planning，您可以完全自定义工作流，从决定您的组织使用哪些对象类型（或记录类型）到配置这些对象如何相互链接。

此外，Workfront Planning可以链接到其他系统中的对象类型，为所有流程创建一个一致的框架。
