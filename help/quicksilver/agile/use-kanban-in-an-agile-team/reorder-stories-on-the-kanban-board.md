# 重新排序 [!UICONTROL 看板] 展示板

您可能想要更改文章在 [!UICONTROL 看板] 展示板。

>[!NOTE]
>
>您不能更改文章在积压文件中的显示顺序 [!UICONTROL 看板] 团队。

文章在 [!UICONTROL 看板] 董事会未指明优先级。 但是，通过让故事更加显眼，它可以影响人们所认为的优先事项。

默认情况下，文章按字母顺序显示在 [!UICONTROL 看板] 展示板。

对 [!DNL Kanban] 展示板，则您所做的任何更改都会保存，因此在您或其他用户下次查看时，所做的更改会保留 [!UICONTROL 看板] 展示板。 （清除浏览器缓存时，您所做的更改不会恢复。）

## 重新排序 [!UICONTROL 看板] 展示板

要更改顺序或文章，请执行以下操作：

1. 单击 **[!UICONTROL 主菜单]** 图标 ![](assets/main-menu-icon.png) 的右上角 [!DNL Adobe] Workfront，然后单击 **[!UICONTROL 团队]**.

1. （可选）单击 **[!UICONTROL 切换组]** 图标 ![切换团队图标](assets/switch-team-icon.png)，然后选择新 [!UICONTROL 看板] 团队，或在搜索栏中搜索团队。

1. 转到 [!UICONTROL 看板] 包含要重新排序的文章的展示板。
1. 将文章图块拖到 [!UICONTROL 看板] 展示板。
