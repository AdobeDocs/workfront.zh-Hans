# [!UICONTROL 讨论区]概述

[!DNL Adobe Workfront] [!UICONTROL 讨论区]是灵活的工具，通过它可访问包含列和卡片的共享讨论区，从而允许团队协作。

显示任务状态是讨论区的常见用法，但您可以将任何项目组合在一起。 讨论区中的卡片可以代表一个人、一群人、一个想法、一个概念或者您所需的任何类型的项目。

使用讨论区，您可以：

* 快速设置任务板
* 配置列以显示状态或类别
* 使用开放式卡（临时卡或连接到[!DNL Workfront]任务和问题的卡）自定义工作流程
* 在一个位置组织和管理团队的工作
* 使用敏捷计划工具，如看板板的登记列

有关如何使用基本讨论区功能的信息，请参阅[讨论区快速入门：文章索引](../agile/get-started-with-boards/get-started-with-boards.md)。 有关讨论区中Agile规划工具的信息，请参阅[Agile规划工具概览](/help/quicksilver/agile/use-boards-agile-planning-tools/agile-planning-tools-overview.md)。

有关在项目中使用讨论区的信息，请参阅[在Agile视图中管理项目](/help/quicksilver/manage-work/projects/manage-projects/manage-projects-in-agile-view.md)。

您还可以在移动应用程序中访问讨论区。 有关信息，请参阅[[!DNL Adobe Workfront] [!UICONTROL 移动版]](/help/quicksilver/workfront-basics/mobile-apps/using-the-workfront-mobile-app/mobile-boards.md)。

有关讨论区功能的视频演练，请参阅[[!DNL Adobe Workfront] [!UICONTROL 讨论区]视频演示](/help/quicksilver/agile/get-started-with-boards/boards-video-demonstrations.md)。

有关[!DNL Workfront]中其他Agile工具的信息，请参阅[Agile概述](../agile/agile-overview.md)。
