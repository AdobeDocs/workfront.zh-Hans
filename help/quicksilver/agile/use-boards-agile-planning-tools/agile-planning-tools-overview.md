# 敏捷规划工具概述：文章索引

<!-- Audited: 5/2025 -->

使用即席卡和连接的卡创建讨论区时，可以利用附加的Agile计划工具。

对看板有用的工具包括：

* 列的WIP限制
* 将卡片分组到泳道中
* 可用于积压工作的录入列

对Scrum有用的工具包括：

* 工作流程和迭代讨论区
* 卡列表或积压工作
* 迭代量度

>[!IMPORTANT]
>
>工作流及其相关工具仅适用于特定的客户组。

您可以按照适合您和团队的方式使用其中任何或所有工具，而无需遵循特定的Agile方法。
