# 敏捷规划工具概述

在使用临时卡片和连接卡片创建展示板时，您可能希望利用其他灵活规划工具，包括列WIP限制、将卡片分组到游泳道，以及可用于积压工作的进纸列。

您可以以适合您和您的团队的方式使用任何或所有这些工具，而无需提交特定的灵活方法（如看板或清理）。
