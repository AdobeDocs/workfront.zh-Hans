# 校对和Creative Cloud Express入门


我们很高兴能介绍Proofing与Creative Cloud Express之间的新集成。 通过此集成，您可以

* 使用Workfront审批模板定义审批工作流

* 在验证查看器中查看Creative Cloud Express草稿内容

* 请参阅Creative Cloud Express中的审核决策，以获得最终批准和发布

## 先决条件

* Workfront和Creative Cloud Express必须部署到同一IMS组织。

* 为使集成正常工作，用户只能在IMS组织内的一个Workfront实例中处于活动状态并启用

* 需要在Unified Shell上启用Workfront实例

* Workfront设置部分已切换为连接到验证（见下文）
