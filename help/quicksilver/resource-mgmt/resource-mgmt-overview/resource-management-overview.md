# 资源管理

资源管理允许资源经理进行业务决策，确保当前可用的单个员工预算与最高优先级的工作预算一致。

虽然资源管理可能从工作角色级别开始，但最终涉及用户、用户分配到实际工作、用户利用率不足或过度承诺等问题。
