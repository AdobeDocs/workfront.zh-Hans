# 资源管理

资源管理允许资源管理人员做出业务决策，以确保现在可用的单个员工按照最高优先工作的预算进行预算。

虽然它可能从职务角色级别开始，但最终资源管理部门会关注用户、他们对实际工作的分配以及他们的利用率不足或过度投入。
