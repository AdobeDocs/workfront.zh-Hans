# 品牌 [!DNL Workfront Proof] 站点 — 高级

>[!IMPORTANT]
>
>本文介绍独立产品中的功能 [!DNL Workfront Proof]. 有关内部校对的信息 [!DNL Adobe Workfront]，请参阅 [校对](../../../review-and-approve-work/proofing/proofing.md).

高级品牌策略在Select和Premium计划中可用，并包含在计划成本中。

有关基本品牌策略（包括登录页面、电子邮件通知和校样）的信息，请参阅 [品牌 [!DNL Workfront Proof] 网站](../../../workfront-proof/wp-acct-admin/branding/brand-wp-site.md).

高级品牌选项包括自定义以下区域：

* 标题和文本颜色
* Web应用程序标题
* 菜单栏和文本颜色
* 功能板欢迎框和快速启动框
* 页脚文本
* 法维克
* 页面标题
* 帮助链接

以下各节更详细地描述高级品牌策略：

## 高级品牌策略概述

你会找到 [!UICONTROL 品牌策略配置] 的 [帐户设置](https://support.workfront.com/hc/en-us/sections/115000912147-Account-Settings) 页面。 要将更改应用到您的帐户，请确保将“品牌”选项设置为 [!UICONTROL 已启用] (1)。
![Advanced_Branding.png](assets/advanced-branding-350x618.png)

有关如何配置高级品牌选项(2-14)的更多详细信息，请参阅下面的部分。

## 高级品牌策略配置

您可以为 [!DNL Workfront Proof]:

* [Web 应用程序](#web-application)
* [标头](#header)
* [标题链接](#header-links)
* [侧栏](#sidebar)
* [欢迎框](#welcome-box)
* [节标题](#section-headers)

### Web 应用程序 {#web-application}

![Web_branding_image.png](assets/web-branding-image-350x181.png)

对于Web应用程序标题(2)，您可以选择三个品牌选项：

* 标题中的品牌图像
* 您的帐户名称在标题中
* 如果禁用品牌策略 [!DNL Workfront Proof] 徽标将保留在web应用程序的标题中

品牌图像 — 最大大小为550x90像素。 您可以使用JPG、GIF或PNG，并且支持透明背景。

帐户名称 — 从帐户详细信息中获取，使用白色字体显示。 您的帐户名称最多可包含60个字符（包括空格和标点符号）。

### 标头 {#header}

在此字段(3)中，您可以设置标题的背景，并且可以选择纯色或背景图像。

颜色 — 与在颜色方案配置的所有其他品牌字段中一样，您可以输入您选择的十六进制颜色值或使用方便易用的拾色器（单击文本字段会打开弹出窗口）。 默认的标题背景颜色为#232d2e。

![Branding_-_color_picker.png](assets/branding---color-picker-350x131.png)

背景图像 — 可以与“品牌”图像结合使用。

* 您可以使用JPG、GIF或PNG — 对于具有透明度的文件，将显示白色背景颜色。
* 标题的高度为96像素，且上传的图像不会针对 [!UICONTROL 不重复] 选项。
* 背景图像位于左上角。

### 标题链接 {#header-links}

在此字段(4)中，您可以在帐户右上角显示的“标题”菜单中修改链接的用户名颜色和颜色。
![Header_links.png](assets/header-links-350x140.png)

### 侧栏 {#sidebar}

选择菜单栏(5)和菜单字体(6)的颜色以自定义您的侧栏

![Menu_bar.png](assets/menu-bar-350x490.png)

>[!NOTE]
>
>将鼠标悬停在选定的菜单栏颜色中，可通过添加恒定的十六进制值来自动调整颜色。

无法自定义New校样按钮颜色。

### 欢迎框 {#welcome-box}

在此字段(7)中，您可以设置“功能板”页面上显示的欢迎框颜色。
![Welcome_box.png](assets/welcome-box-350x110.png)

### 节标题 {#section-headers}

利用这些字段，可自定义 [[!UICONTROL 帐户设置]](https://support.workfront.com/hc/en-us/sections/115000912147-Account-Settings) 页面。
![Section_headers.png](assets/section-headers-350x138.png)

**页脚**

在品牌策略配置的此字段(10)中，您可以撰写将显示在所有帐户页面底部的页脚。 您可以使用内置的WYSIWYG编辑器，也可以简单地粘贴您自己的设计。
![Footer.png](assets/footer-350x157.png)

>[!NOTE]
>
>无法在页脚文本编辑器中编辑HTML，但可以粘贴复制的设计（包括所有链接和图像）。

**收藏夹和页面标题**

您可以自定义 [!DNL Workfront Proof] 页面通过设置您自己的收藏夹图像（.ICO文件）(11)和页面标题(12)在浏览器中显示 — 这些页面将显示在所有帐户页面的浏览器选项卡/窗口标题中。
![Favico_and_Page_title.png](assets/favico-and-page-title-350x95.png)

>[!NOTE]
>
>您的收藏夹图标也会显示在 [!DNL Workfront Proof] 查看器窗口，用于帐户中创建的所有校样。

**帮助链接**

您可以自定义自己的帮助链接，以将用户和审阅人指向您自己的内容。 要激活此集，请 [!UICONTROL 帮助链接] 选项(13)，并在下一个字段中添加您的链接(14)。 您最多可以配置四个链接，它们将可用：

* 在 [!UICONTROL 标题] 菜单 [!UICONTROL 帮助] 链接
   ![Help_links1.png](assets/help-links1-350x170.png)

* 在 [!UICONTROL 帮助] 面板 [!DNL Workfront Proof] [!UICONTROL 查看器] 侧栏
   ![Help_links2.png](assets/help-links2-350x532.png)

企业计划客户还可以选择完全自定义以下内容，但需额外付费：

* 完全自定义您的登陆页面（例如，登录和注销页面、忘记密码页面）
* 完全自定义域

请通过sales.team@workfront.com与我们联系，以了解有关其他品牌策略选项的更多信息。
