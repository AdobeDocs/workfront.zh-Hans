# 已删除旧版验证查看器

>[!IMPORTANT]
>
>本文提及独立产品[!DNL Workfront Proof]中的功能。 有关[!DNL Adobe Workfront]内部校对的信息，请参阅[校对](../../../review-and-approve-work/proofing/proofing.md)。

在2018.3版本之前，支持旧版验证查看器。 在2018.3版本（2018年11月）中，旧版验证查看器和所有其他依赖于Flash的应用程序已被删除并且不再可用。
