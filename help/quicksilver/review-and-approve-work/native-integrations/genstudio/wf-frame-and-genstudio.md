# GenStudio for Performance Marketing与Frame.io集成快速入门

通过集成GenStudio for Performance Marketing和Frame.io，您可以

* 使用Workfront统一审批模板定义审阅和审批工作流

* 在Frame.io查看器中审阅和批准GenStudio for Performance Marketing草稿内容

* 在GenStudio for Performance Marketing中查看审核决策以进行最终批准和发布

## 集成要求

* Workfront和GenStudio for Performance Marketing必须部署到同一个Identity Management system (IMS)组织。

* Workfront实例必须启用Adobe企业存储。 有关详细信息，请参阅[启用Adobe企业存储](/help/quicksilver/administration-and-setup/set-up-workfront/configure-system-defaults/enable-esm.md)。


## 使用Workfront统一审批模板定义审批工作流

如果贵组织的内容审核流程经常重复或由同一人员审核，则可以使用统一的审批模板来自动执行审核和审批工作流。

### 在Workfront中创建统一的审批模板

可以为仅一个或两个审阅人创建简单的单阶段模板，也可以为具有许多阶段和依赖关系的复杂审阅创建多阶段模板。

有关在Workfront中创建审批工作流模板的详细信息，请参阅[为文档创建审批工作流模板](/help/quicksilver/review-and-approve-work/document-reviews-and-approvals/manage-document-approvals/create-approval-template.md)。

### 在GenStudio for Performance Marketing中选择或修改模板

当用户在GenStudio for Performance Marketing中启动审核时，他们只需选择所需的模板即可。 用户可以随时轻松更改任何统一的审批审批工作流模板，添加或删除审阅人和阶段。

有关详细信息，请参阅[请求审阅和批准](https://experienceleague.adobe.com/zh-hans/docs/genstudio-for-performance-marketing/user-guide/approve/request-review)。

## 在Frame.io查看器中审阅和批准GenStudio for Performance Marketing草稿内容

您可以在Frame.io查看器中直接在GenStudio for Performance Marketing中查看和批准草稿内容。

使用Frame.io查看器，您可以

* 发表评论
* 标记草稿，显示需要更改的内容
* 做出决定

有关使用Frame.io查看器的详细信息，请参阅[在媒体上发表评论](https://help.frame.io/en/articles/9105251-commenting-on-your-media)。


## 在GenStudio for Performance Marketing中查看审核决策以进行最终批准和发布

资产完成审阅和批准流程后，您可以直接从GenStudio for Performance Marketing查看审阅决策并发布内容。

有关详细信息，请参阅[发布批准的内容](https://experienceleague.adobe.com/zh-hans/docs/genstudio-for-performance-marketing/user-guide/approve/publish-content)。
