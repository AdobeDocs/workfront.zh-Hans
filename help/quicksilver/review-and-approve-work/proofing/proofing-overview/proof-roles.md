# 校样角色概述

利用校样角色，可授予对单个校样的权限。 校样角色与校样权限配置文件不同。 有关校样权限配置文件的更多信息，请参阅 [校样权限配置文件概述](../../../review-and-approve-work/proofing/proofing-overview/permission-profiles.md).

## 验证角色

在邀请用户查看校样时，会为用户授予单独校样的校样角色。 如果您拥有Workfront Pro+计划，则可以向具有“作品”或“计划”许可证的用户授予校对角色。 如果您拥有Workfront Premium旧版计划，则可以向具有指定校对许可证的用户授予校对角色。 有关更多信息，请参阅 [访问Workfront中的校对功能](../../../administration-and-setup/manage-workfront/configure-proofing/access-to-proofing-functionality.md).

>[!NOTE]
>
>新Workfront计划的用户可以向系统中的任何用户授予作者或审核者角色。 旧版计划中的用户可以向系统中具有校样许可证的任何用户授予作者或审核者角色。

校样角色定义审阅人可以对该特定校样执行的操作。

**示例：** 如果您被分配了“审阅者”角色，系统将要求您通过添加标记和注释来审阅校样，但您没有批准或拒绝校样的选项。

某些校样角色为审阅人提供了校样的编辑权限（即使其帐户配置文件没有），并允许他们使用一些其他功能，例如添加对注释的操作、创建新版本以及向校样添加更多审阅人。
