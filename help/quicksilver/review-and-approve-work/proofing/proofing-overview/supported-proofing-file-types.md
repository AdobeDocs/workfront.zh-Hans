# 支持的校对文件类型和大小限制概述

您可以与审阅人共享和审阅各种类型的文件，即使他们没有在其计算机上安装用于创建文件的软件(如Photoshop或Illustrator)。 我们支持静态、视频、音频、Web捕获和交互式文件类型，大小最大为4 GB。

## 文件大小限制

Workfront具有以下限制：

* 每个文件必须小于4 GB。
* XLS和XLSX文件必须小于100 MB。
* 附加到校样注释的文件必须小于30 MB。
* PNG的高度和宽度必须小于65535像素。
* 静态组合校样最多可支持50个文件。
* 上传为校样的文档必须少于2,000页。

>[!NOTE]
>
>较大的文件需要更长时间才能上传，如果连接不稳定，则操作可能会终止/中断。

## Zip文件

出于安全原因，我们不支持或解压ZIP文件中的文件。 在上传文件之前解压缩文件。 如果不执行此操作，则上传不会失败。

仅当zip文件包含完整的交互式项目并且您的帐户启用了交互功能（高级计划）时，才能提交zip文件。

有关更多信息，请参阅 [交互式内容校样概述](../../../review-and-approve-work/proofing/proofing-overview/interactive-content-proofs.md).

## InDesign和Quark文件

我们不支持InDesign和Quark文件，因为它们不是自包含的文件类型，并且它们引用了许多其他文件，如图像和字体。 这些文件类型的输出因输出设置而异。 我们建议客户将InDesign和Quark文件另存为PDF并上传PDF。
