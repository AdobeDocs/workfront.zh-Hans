# 在桌面校对查看器中查看非交互式内容

虽然桌面校对查看器专为审阅交互式内容而设计，但您也可以使用它来审阅静态和视频内容。 您的Adobe Workfront管理员可以将桌面验证查看器配置为每次启动静态验证或视频验证时打开。

在桌面验证查看器中打开非交互式验证后，它的工作方式与Web验证查看器的工作方式相同。 如果您需要有关此功能的信息，请参阅[查看校对](../../../review-and-approve-work/proofing/reviewing-proofs-within-workfront/review-a-proof/review-a-proof.md)。
