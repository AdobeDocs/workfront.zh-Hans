# 删除了各种单点登录选项

Adobe Workfront删除了对各种单点登录(SSO)选项的支持。 最终，在所有Workfront产品和服务中提供一致、更安全的登录体验。

2020年，从产品中删除了以下单点登录选项：

* LDAP
* Active Directory
* SAML 1.1

有关可用的SSO解决方案的信息，请参阅 [Adobe Workfront中的单点登录概述](../../add-users/single-sign-on/sso-in-workfront.md).
