# 删除各种单点登录选项

Adobe Workfront已取消对各种单点登录(SSO)选项的支持。 最终结果是所有Workfront产品和服务都可获得一致且更安全的登录体验。

2020年从产品中删除了以下SSO选项：

* LDAP
* Active Directory
* SAML 1.1

有关可用SSO解决方案的信息，请参阅[Adobe Workfront中的单点登录概述](../../add-users/single-sign-on/sso-in-workfront.md)。
