# 允许从Adobe Workfront应用程序发送电子邮件

如果您已将防火墙或邮件服务器配置为仅允许访问特定供应商，则需要将某些Adobe Workfront IP地址添加到防火墙允许列表。 有关说明，请参阅 [配置防火墙的允许列表](../../../administration-and-setup/get-started-wf-administration/configure-your-firewall.md).
