# 更改用户名

您可以通过编辑用户的用户配置文件来编辑其用户名（电子邮件地址）。 有关说明，请参阅[编辑用户配置文件](../../../administration-and-setup/add-users/create-and-manage-users/edit-a-users-profile.md)中有关“电子邮件地址”字段的信息。
