# 组：文章索引

Workfront管理员可以创建与您的部门结构一致的用户组。 分组与类似，但与团队和公司不同。

Workfront管理员向组授予对需要工作和沟通的Workfront区域的访问权限。 然后，每个组可以将其Workfront信息（如用户、模板、自定义表单和项目）与其他部门的那些信息分开。
