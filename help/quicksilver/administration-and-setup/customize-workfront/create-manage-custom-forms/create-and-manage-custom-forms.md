# 自定义表单：文章索引

<!-- Audited: 1/2024 -->

您可以构建自定义表单，以便用户将其附加到Workfront对象。 处理对象的用户可以填写自定义表单以提供有关对象的信息。

您可以在Workfront实例中创建的自定义表单的数量没有限制。 您可以附加到对象的自定义表单限制为10个。
