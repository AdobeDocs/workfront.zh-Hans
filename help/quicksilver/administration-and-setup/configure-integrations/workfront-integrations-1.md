# [!DNL Adobe Workfront] 集成

您可以将Adobe Workfront与第三方应用程序集成。 集成可以扩展Workfront的实用程序，并根据您组织的需求对其进行定制。

集成 [!DNL Workfront] 分为以下类别：

## 内置（本机）集成

[!DNL Workfront] 提供了各种集成，您可以直接从Workfront应用程序进行配置，或通过安装直接从其他应用程序进行配置 [!DNL Workfront] 该应用程序的加载项。

其中一些集成可免费获得，而其他集成则需要额外购买：

* [内置（本机）集成快速入门](#get-started-with-built-in-native-integrations)
* [集成可供使用，无需额外付费](#integrations-available-at-no-additional-cost)
* [需要额外购买的集成](#integrations-requiring-an-additional-purchase)

### 内置（本机）集成快速入门 {#get-started-with-built-in-native-integrations}

要开始使用内置（本机）集成，请按照以下工作流程操作：

1. **确定最适合您需求的本机集成**

   在选择集成时，请务必选择用户已经定期使用的应用程序。 集成 [!DNL Workfront] 使用经常或经常使用的应用程序，将提高用户的采用率。

   有关可用集成的列表，请参阅 [集成可供使用，无需额外付费](#integrations-available-at-no-additional-cost) 和 [需要额外购买的集成](#integrations-requiring-an-additional-purchase).

1. **识别技术管理员**

   为要与集成的应用程序确定贵组织的技术管理员 [!DNL Workfront]. 例如，如果您选择与集成 [!DNL Jira]，则必须连接您公司的 [!DNL Jira] 管理员，并让他们参与其平台中的任何配置需求。 您还应该让此管理员参与与 [!DNL Workfront] 客户支持。

   要开始使用管理员，您需要向他们提供该集成的具体说明。 有关这些说明的链接，请参阅 [集成可供使用，无需额外付费](#integrations-available-at-no-additional-cost).

1. **参与 [!DNL Workfront] 客户支持**

   如果您的管理员在本机集成设置中遇到任何问题，您可以与 [!DNL Workfront] 客户支持，以帮助指南或进行故障诊断。 While [!DNL Workfront] 不会为您设置集成，我们承诺为您提供自行完成集成所需的所有必要工具和资源。 您可以直接与客户支持部门联系，也可以浏览大量资源文章以帮助您轻松设置集成。

   要联系客户支持，请参阅 [联系客户支持](../../workfront-basics/tips-tricks-and-troubleshooting/contact-customer-support.md).

   有关帮助文章的链接，请参阅 [集成可供使用，无需额外付费](#integrations-available-at-no-additional-cost). 和 [需要额外购买的集成](#integrations-requiring-an-additional-purchase).

   有关创建支持案例的说明，请参阅 [联系客户支持](../../workfront-basics/tips-tricks-and-troubleshooting/contact-customer-support.md).

   >[!NOTE]
   >
   >您必须是贵组织的分配支持联系人才能提交支持案例。 提交案例后，您可以将另一个系统的技术管理员添加到对话中。 他们不能自己直接联系支持人员。

我们了解，可用的内置（本机）集成可能无法满足您公司的所有需求。 如果出现这种情况，请联系您的客户经理来讨论自定义集成。

### 需要额外购买的集成 {#integrations-requiring-an-additional-purchase}

* [!DNL Workfront DAM]

   有关更多信息，请参阅 [[!DNL Adobe Workfront DAM] within [!DNL Adobe Workfront]](../../documents/workfront-dam-within-workfront/workfront-dam-in-workfrontt.md).

* [!DNL Workfront Fusion]

   有关更多信息，请参阅 [[!DNL Adobe Workfront Fusion] 概述](../../workfront-fusion/get-started/workfront-fusion-overview.md).

* Adobe Experience Manager Assets
   [!DNL ]
有关更多信息，请参阅 [配置 [!DNL Workfront] with [!DNL Adobe Experience Manager] 旧连接器](../../administration-and-setup/configure-integrations/configure-workfront-aem.md).

## 合作伙伴集成

[!DNL Workfront] 拥有丰富的合作伙伴生态系统，其中包含大量可丰富您 [!DNL Workfront] 体验。

现有合作伙伴产品的示例包括与 [!DNL Salesforce.com], [!DNL Oracle Responsys]和 [!DNL Microsoft Visual Studio]. 有关这些集成以及其他令人兴奋的集成的更多信息，请参阅 [[!DNL Workfront] Marketplace](https://www.workfront.com/enterprise/marketplace/apps/).

## [!DNL Workfront] API

的 [!DNL Workfront] API允许您扩展和增强 [!DNL Workfront] 体验。 您可以使用我们的开放API创建您自己的集成。 有关 [!DNL Workfront] API，请参阅 [!DNL Workfront] [API Explorer](../../wf-api/general/api-explorer.md).
