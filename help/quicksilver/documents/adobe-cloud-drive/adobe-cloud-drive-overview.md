# Adobe Cloud Drive概述

Adobe Cloud Drive是一款桌面应用程序，可将您的Adobe云存储项目作为驱动器装载到Mac或Windows计算机上。 Workfront项目显示为文件夹，您可以使用任何桌面应用程序直接在Adobe云存储上打开、创建、编辑和保存文件到Workfront项目。

Adobe Cloud Drive中的文件与您在Workfront和Frame.io中看到的文件相同，因此您在桌面上所做的更改在两个产品的连接的Workfront项目中均可用。

[观看有关Adobe Cloud Drive中可用功能的视频概述。](https://video.tv.adobe.com/v/3492492){target="_blank"}

有关Adobe Cloud Drive连接到的存储的更多信息，请参阅[Adobe Cloud Storage概述](/help/quicksilver/review-and-approve-work/esm-overview.md)。

## 使用Adobe Cloud Drive可以做什么

* 在Finder (Mac)或文件资源管理器(Windows)中作为文件夹从Adobe云存储访问项目。
* 在Adobe Creative Cloud应用程序、Microsoft Office或计算机上安装的任何应用程序中打开、创建和编辑文件。
* 将更改保存在本地，然后让Adobe Cloud Drive将它们同步到Adobe Cloud Storage上的项目。
* 在没有网络连接的情况下旅行或工作时，将文件标记为脱机访问。 恢复连接后，更新将同步到项目中的文件，然后可以从本地缓存中删除这些文件以释放磁盘空间。
* 与团队成员协作处理相同的项目文件。

## 谁可以使用Adobe Cloud Drive

要使用Adobe Cloud Drive，您的组织必须位于已启用Adobe云存储的工作流Ultimate包上。

只有使用Adobe云存储的项目会显示在驱动器中。 旧版Workfront项目不会显示。 有关差异的更多信息，请参阅[Adobe云存储与旧版Workfront存储之间的差异](/help/quicksilver/review-and-approve-work/esm-overview.md#differences-between-adobe-cloud-storage-and-legacy-workfront-storage)。

如果您没有看到可供下载的Adobe Cloud Drive，或者您的组织在为您管理应用程序安装，请与Workfront管理员联系。
