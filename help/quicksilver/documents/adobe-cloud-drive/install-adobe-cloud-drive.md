# 安装Adobe云驱动器

要在桌面上使用Adobe云存储项目，请在Mac或Windows计算机上安装Adobe云驱动器。

有关什么是Adobe Cloud Drive以及您可以对其执行什么操作的信息，请参阅[Adobe Cloud Drive概述](/help/quicksilver/documents/adobe-cloud-drive/adobe-cloud-drive-overview.md)。

## 先决条件

在安装Adobe Cloud Drive之前，请确保满足以下条件：

* 您的设备符合本文[系统要求](#system-requirements)部分中列出的系统要求。
* 您的组织使用的是Workfront版本，该版本支持Adobe云存储。

如果贵组织在为您管理应用程序安装，但您看不到可供下载的Adobe云驱动器，请与Workfront管理员联系。

## 系统要求

**操作系统：**

* macOS 13.5或更高版本（英特尔和Apple Silicon）
* Windows 11 22H2 (22621.6060)或更高版本(x64)
* Windows 11 23H2 (22631.6199)或更高版本(ARM)

**硬件：**

* 最小4 GB RAM
* 500 MB可用磁盘空间供应用程序使用
* 为缓存的文件增加空间（因使用情况而异）

**网络：**

* 用于初始设置和同步的Internet连接
* 为提升性能，建议使用稳定连接

要确保Adobe Cloud Drive可以访问它依赖的服务，请允许[Adobe网络端点](https://helpx.adobe.com/in/enterprise/kb/network-endpoints.html)中列出的网络端点。

## 下载安装程序

1. 转到[Adobe Software Distribution](https://experience.adobe.com/#/downloads)。

1. 使用您的 Adobe ID 登录。
1. 在&#x200B;**常规**&#x200B;下载列表中，找到&#x200B;**Adobe云驱动器**。
1. 选择操作系统的安装程序：

   * **macOS：**&#x200B;适用于Mac（ARM64或英特尔）的Adobe Cloud Drive
   * **Windows：**&#x200B;适用于Windows （x64或ARM64）的Adobe云驱动器

1. 单击&#x200B;**“下载”。**

   >[!IMPORTANT]
   >
   >如果您没有看到Adobe Cloud Drive安装程序，则表示您的组织可能未使用支持Adobe Cloud Storage的Workfront版本。 联系Workfront管理员。

## 安装Adobe云驱动器

请按照适用于您的操作系统的步骤进行操作。

### 在macOS上安装

1. 打开下载的`.dmg`文件。
1. 双击显示的安装图标。
1. 按照屏幕上的安装向导进行操作。

1. 出现提示时，使用您的Adobe ID登录。

   如果另一个Adobe桌面应用程序已在运行并且您已登录，则Adobe Cloud Drive会自动使用该活动会话。

### 在Windows上安装

1. 运行下载的`.exe`安装程序。

1. 按照安装向导中的提示操作。
1. 出现提示时，单击&#x200B;**安装**。
1. 从“开始”菜单启动&#x200B;**Adobe Cloud Drive**。
1. 出现提示时，使用您的Adobe ID登录。

## 验证安装

安装后，Adobe Cloud Drive在计算机上显示为已安装的驱动器：

* **macOS：**&#x200B;位于&#x200B;**位置**&#x200B;下的Finder侧边栏中，作为&#x200B;**Adobe Cloud Drive**。
* **Windows：**&#x200B;在&#x200B;**此电脑**&#x200B;下的文件资源管理器中，作为&#x200B;**Adobe云驱动器**。

您应该看到Adobe云存储项目在驱动器中作为文件夹列出。

有关在驱动器中使用项目的信息，请参阅[使用Adobe云驱动器](/help/quicksilver/documents/adobe-cloud-drive/use-adobe-cloud-drive.md)。
