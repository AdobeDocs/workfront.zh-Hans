# 将链接复制并共享到文档文件夹

您可以复制以下任何内容中包含的任何文档文件夹的直接链接 [!DNL Workfront] 对象： [!UICONTROL 项目], [!UICONTROL Portfolio], [!UICONTROL 项目], [!UICONTROL 任务]或 [!UICONTROL 问题]. 无法复制 [!UICONTROL 文档] 区域，因为这些文件夹直接链接到您的用户配置文件，无法与其他人共享。

## 将链接复制并共享到文档文件夹

要将链接复制到文档文件夹，请执行以下操作：

1. 导航到 [!DNL Workfront] 包含文档文件夹的对象。
1. 单击 **[!UICONTROL 更多]** 菜单，然后选择 **[!UICONTROL 复制URL]**. 您可以共享此链接以便快速访问文件夹。 用户必须至少具有查看对象的访问权限才能查看文件夹。
   ![复制文档文件夹url](assets/copy-doc-folder-url.png)
