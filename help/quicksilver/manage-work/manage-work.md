# 管理工作信息

您可以使用Adobe Workfront管理您的工作。

您可以首先确定用于定义工作的工作对象，具体取决于其复杂性和您分配给它的资源或预算。

本节介绍如何将您的工作组织到较大的工作项（如项目、项目组合或项目）中，以及如何将较大的工作量划分为更易于管理的工作项（如任务和问题）。

此外，您可以使用甘特图定义工作的时间线，或在没有时间完成工作时将工作委派给其他人。
