# 创建请求

<!--
{{highlighted-preview}}
-->

请求和问题在Adobe Workfront中可互换。 您可以通过执行以下操作之一来创建请求：

* 在Workfront中提交请求
* 在其他应用程序中提交请求
* 将请求通过电子邮件发送到设置为请求队列的项目。

所有请求都会显示在它们已提交项目的“问题”区域中。
