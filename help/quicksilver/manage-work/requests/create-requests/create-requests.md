# 创建请求

<!--
{{highlighted-preview}}
-->

在Adobe Workfront中，请求和问题可以互换。 您可以通过执行以下操作之一来创建请求：

* 在Workfront中提交请求
* 在其他应用程序中提交请求
* 通过电子邮件将请求发送到设置为请求队列的项目。

所有请求都将显示在其已提交项目的问题区域中。
