# 请求项目

您可以在开始处理项目之前，先请求其批准。 这会创建一个状态为“构思”的项目。 我们建议为您的项目构建一个业务案例，以概要介绍您希望通过完成该项目来实现的一些要点，并向您的管理层介绍您的人工和费用预算。

在Adobe Workfront的以下区域添加新项目时，您可以请求一个项目：

* 在“项目”区域
* 在Portfolio的“项目”区域
* 在方案的“项目”区域
* 当您是组管理员时，在项目组的“组”区域中。

## 请求项目

1. 执行下列操作之一：

   * 单击 **主菜单** ![](assets/main-menu-icon.png)，单击 **项目**，然后展开 **新建项目**.
   * 转到项目组合，然后展开 **新建项目**.
   * 转到项目，然后展开 **新建项目**.
   * 如果您是群组管理员，则还可以在您管理的群组的“项目”部分创建项目。 有关更多信息，请参阅 [创建和修改组的项目](../../../administration-and-setup/manage-groups/work-with-group-objects/create-and-modify-a-groups-projects.md).

1. 单击 **请求项目**.

   ![](assets/new-project-dropdown-nwe-350x358.png)

1. 执行下列操作之一：

   * 按照与在定义业务案例时创建项目相同的步骤操作。 （推荐）

      有关定义业务案例和请求项目的信息，请参阅 [为项目创建业务案例](../../../manage-work/projects/define-a-business-case/create-business-case.md).

   * 从头开始创建项目或使用模板，然后将状态更改为 **请求**.

      有关创建项目的信息，请参阅 [创建项目](../../../manage-work/projects/create-projects/create-project.md) 或 [使用模板创建项目](../../../manage-work/projects/create-projects/create-project-from-template.md).
   您请求的项目的状态为 **请求** 它显示在 **请求的项目** 选项卡(如果您将其与Portfolio关联)。 现在，项目组合经理可以审核项目组合，批准或拒绝项目组合，在这种情况下，项目状态将更改为 **已批准** 或 **被拒绝**.
