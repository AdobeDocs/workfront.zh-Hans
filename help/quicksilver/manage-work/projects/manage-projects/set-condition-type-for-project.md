# 设置项目的条件类型

作为项目经理，您可以通过更新项目的条件类型来确定如何计算项目的条件。 项目条件是项目进展情况的直观表示。

## 为项目设置条件类型

1. 转到要更新条件类型的项目。
1. 执行以下操作之一： 

   * 单击 **更多** 菜单 ![](assets/qs-more-menu.png) 在项目名称的右侧，单击 **编辑**.
   * 单击 **项目详细信息** 中。

   ![](assets/update-condition-type-nwe-350x108.png)

1. 在 **条件类型** 字段中，选择以下选项之一：

   * **手动：** 项目所有者可手动设置项目的条件。

      在这种情况下，项目所有者可以在项目标题或项目详细信息部分中更新项目的条件。

   * **进度状态：** Workfront根据项目的进度状态设置条件。 有关如何计算进度状态的信息，请参阅 [项目进度状态概述](../../../manage-work/projects/planning-a-project/project-progress-status.md).

1. 单击 **保存** 在“编辑项目”(Edit Project)框中修改条件类型时。

   单击 **保存更改** 修改“项目详细信息”部分中的条件类型时。
