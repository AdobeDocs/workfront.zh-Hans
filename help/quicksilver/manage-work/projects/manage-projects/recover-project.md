# 恢复已删除的项目

当您需要恢复项目时，请联系Adobe Workfront管理员或组管理员。 他们可以恢复过去30天内已删除的项目、任务、问题和其他项目。 30天后，项目将被永久删除，无法恢复。

有关恢复项目的更多信息，请参阅 [恢复已删除的项目](../../../administration-and-setup/manage-workfront/manage-deleted-items/restore-deleted-items.md).
