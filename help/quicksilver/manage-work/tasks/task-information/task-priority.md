# 更新任务优先级

您可以为任务定义优先级，以指示任务的重要性。

您必须具有任务的“管理”权限才能修改任务的“优先级”。

## 任务优先级概述

使用任务优先级时请考虑以下事项：

* Adobe Workfront管理员可以修改Workfront中的优先级标签，也可以创建新的优先级。 因此，您在Workfront实例中对“优先级”字段可能具有的选项可能与下面列出的选项有所不同。 有关自定义实例中优先级名称的信息，请参阅 [创建和自定义优先级](../../../administration-and-setup/customize-workfront/creating-custom-status-and-priority-labels/create-customize-priorities.md).
* 每个优先级标签都有一个与其关联的编号，该编号无法修改。

   您应该熟悉组织中使用的优先级的编号顺序。

   例如，如果已修改优先级的标签，则贵组织可能使用编号1来表示紧急任务。

   请参阅下表，了解每个默认优先级标签及其相应编号：

   | **优先级标签（可修改）** | **优先级编号（无法修改）** |
   |---|---|
   | 无 | 0 |
   | 低 | 1 |
   | 正常 | 2 |
   | 高 | 3 |
   | 紧急 | 4 |



## 修改任务的优先级

1. 转到要修改其优先级的任务。
1. 单击 **任务详细信息** 中。
1. 单击 **编辑** 图标 ![](assets/qs-edit-icon.png)，然后在 **概述** ，请为 **优先级** 字段。
1. 单击 **保存** **更改**.
