# 问题持续时间概述

问题的持续时间是问题的计划开始日期与计划完成日期之间的天数。

问题的持续时间由Adobe Workfront自动计算，并且无法手动修改。

您可以在问题报告或问题列表的视图中显示问题的持续时间。

有关生成报告的详细信息，请参阅[创建自定义报告](../../../reports-and-dashboards/reports/creating-and-managing-reports/create-custom-report.md)。
