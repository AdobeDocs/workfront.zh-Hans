# 问题信息：文章索引

<!--Audited: 08/2025-->

在处理项目时，您可能会发现出现意外事件。 您可以将这些意外事件记录为特定项目或任务的问题。

与任务不同，问题不能驱动项目的完成百分比。 但是，问题指可能会阻止项目完成的工作项。 未完成的问题表示在项目可以关闭之前，仍有工作要完成。
