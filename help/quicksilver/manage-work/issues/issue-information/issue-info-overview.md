# 问题信息

在处理项目时，您可能会发现出现意外事件。 您可以将这些意外事件记录为特定项目或任务的问题。

与任务不同，问题无法驱动项目完成的百分比。 但是，问题是可能会阻止项目完成的工作项目。 未完成的问题表示在项目关闭之前仍需要完成工作。
