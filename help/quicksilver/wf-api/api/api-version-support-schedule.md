# API版本控制和支持计划



Adobe Workfront会定期发布新的API版本，通常每年两次。 API版本在发布后受支持3年。 不再支持3年以前的API版本。

有关标识和使用API版本的更多信息，请参阅[在您的集成中指定API版本](/help/quicksilver/wf-api/api/specify-api-version-integrations.md)。
