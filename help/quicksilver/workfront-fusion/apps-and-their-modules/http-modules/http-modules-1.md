# HTTP模块

>[!NOTE]
>
>[!DNL Adobe Workfront Fusion] 需要 [!DNL Adobe Workfront Fusion] 除 [!DNL Adobe Workfront license].

HTTP应用程序提供了各种模块，用于基于超文本传输协议(HTTP)协议进行通信。 HTTP是万维网数据通信的基础。 您可以使用模块下载网页和文件、调用Web挂接和API端点等。

模块的正确选择取决于要访问的资源所使用的身份验证/授权机制。
