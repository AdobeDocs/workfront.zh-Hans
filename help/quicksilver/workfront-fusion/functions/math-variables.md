## [!UICONTROL 随机]

返回该范围内的浮点伪随机数 [`0`,`1`] (包括 `0`，但不是 `1`)。

使用以下公式在范围内生成整数伪随机数 [`min`,`max`] (包括 `min` 和 `max`):

![](assets/math-variable-random-350x61.png)

```
{{floor(random * (1.max - 1.min + 1)) + 1.min}}
```
