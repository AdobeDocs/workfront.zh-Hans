## [!UICONTROL random]

返回范围中的浮点伪随机数 [`0`，`1`] (包括 `0`，但不匹配 `1`)。

使用以下公式可生成范围内的整数伪随机数 [`min`，`max`] (包括两者 `min` 和 `max`)：

![](assets/math-variable-random-350x61.png)

```
{{floor(random * (1.max - 1.min + 1)) + 1.min}}
```
