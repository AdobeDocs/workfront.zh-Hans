# 使用Adobe Workfront内置报告

<!--Audited: 07/2024-->

Adobe Workfront提供了大量可使用的内置报告。

Workfront管理员可以隐藏内置报告，这样用户就无权访问它们。

有关如何隐藏内置报告的详细信息，请参阅[隐藏内置报告](../../../administration-and-setup/manage-workfront/configure-reports/hide-built-in-reports.md)。
