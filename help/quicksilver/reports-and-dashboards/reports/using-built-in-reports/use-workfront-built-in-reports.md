# 使用Adobe Workfront内置报告

<!--Audited: 11/2024-->

Adobe Workfront包含大量可供使用的内置报告。

Workfront管理员可以隐藏内置报表，以便用户无权访问这些报表。 有关如何隐藏内置报表的详细信息，请参阅[隐藏内置报表](../../../administration-and-setup/manage-workfront/configure-reports/hide-built-in-reports.md)。
